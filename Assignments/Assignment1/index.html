Programming languages are what allow humans to easily tell computers what to do. Computers run everything they do in binary, a language consisting only of 1s and 0s. The programing languages that most humans use are much easier to understand. The computer takes these languages and compiles them into binary. Different programing languages are specialized for doing different things better. In this website we will discuss several programing languages, and one markup language.
